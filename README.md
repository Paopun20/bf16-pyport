# BF16 - Python Compiler | **Visual Brainfuck Runtime**

---

## Requirements

- **Python** 3.10+
- **Pip packages**:
  ```bash
  pip install pygame numpy
  ```

## Usage

RGB332 color:
```bash
python -m bf16.py examples/snake.b
```
Grayscale color:
```bash
python -m bf16_grayscale.py examples/badapple.b
```
